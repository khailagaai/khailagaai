# Khai Lagaai Betting Calculator 🏏

A sleek, modern, and perfectly aligned static web application designed to calculate complex Khai (Lay) and Lagaai (Back) sports betting mathematics instantly. Built with pure HTML, CSS, and Vanilla JavaScript—no external dependencies or backend required.

## 🚀 Features

- **Instant Math:** Automatically calculates real-time liabilities and profits across both teams.
- **Dynamic Cashout & Loss Cut:** Smart modal system to help you instantly balance the book or cut your losses with calculated counter-bets.
- **Light & Dark Mode:** Carefully curated, premium aesthetics that switch instantly and save to your local browser storage.
- **Fully Local & Private:** All state and history are saved safely in your browser's `localStorage`. No data is ever sent to a server.
- **Responsive Design:** A beautiful glassmorphism-inspired UI that looks perfect on both desktop and mobile devices.

## 🛠️ Tech Stack
- **HTML5:** Semantic and accessible structure.
- **CSS3:** Custom CSS variables, CSS Grid, Flexbox, and glassmorphism styling without any UI frameworks.
- **JavaScript (ES6):** Vanilla JS handling all DOM manipulation, financial logic, theme management, and asynchronous custom modals.

## 💻 How to Run Locally
1. Clone the repository or download the ZIP.
2. Open `index.html` in any modern web browser.
3. No build steps, no `npm install`, it just works!

## 💡 About
Built to simplify the mental math required for traditional sports betting markets, specifically tailored for Khai/Lagaai systems.
